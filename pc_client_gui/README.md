# 墙体裂缝检测上位机 - 项目文件结构说明

## 📁 目录结构

```
wall_crack_detection/
│
├── main.py                 # 主程序入口，包含顶层UI骨架
├── config.ini              # 系统配置文件（TCP端口、API密钥等）
│
├── data/                   # 数据存储目录
│   └── crack_detection.db  # SQLite数据库文件
│
├── images/                 # 图片存储目录
│   └── {日期}/             # 按日期归档子目录
│       └── {设备ID}/       # 按设备ID归档子目录
│
├── logs/                   # 日志文件目录
│   └── app.log             # 应用运行日志
│
├── reports/                # 报告导出目录
│   ├── excel/              # Excel识别报表
│   └── pdf/                # AI分析PDF报告
│
└── rk3588_client/          # RK3588端测试脚本
    └── test_tcp_client.py  # TCP发送测试脚本
```

## 📦 开发环境要求

- Python 3.10+
- PyQt5
- threading (内置)
- os (内置)
