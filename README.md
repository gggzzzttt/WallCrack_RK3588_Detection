# 墙体裂缝分割分类检测系统（RK3588端侧部署+外网TCP上位机联动）
## 项目简介
本项目为嵌入式机器视觉竞赛作品，基于BiSeNetV2语义分割 + MobileNetV3裂缝分类双轻量化模型，完成建筑墙体缺陷识别；模型量化为RKNN格式部署在RK3588开发板，通过公网TCP网络实时向上位机推送推理效果图、裂缝等级与置信度数据，支持异地外网远程监控检测画面。

## 仓库模块说明（精简核心代码，无冗余缓存文件）
1. **model_BV3-CrackNet_train**
   包含数据集加载、双模型联合训练、ONNX导出、RKNN模型量化转换完整代码；提供预训练权重`model_best.pth`、导出ONNX文件，可复现训练与模型转换全流程。
2. **rk3588_deploy**
   部署推理核心代码`crack_deploy.py`，内置BiSeNet分割、MobileNet分类两个RKNN量化模型和联动运行代码。
3. **pc_client_gui**
   电脑上位机客户端程序，主动TCP连接开发板公网IP，实时打印裂缝检测文字结果、弹窗展示自适应缩放大字可视化效果图。

项目创新点
1、双模型轻量化并行推理：BiSeNetV2 高精度裂缝分割 + MobileNetV3 缺陷等级分类，兼顾精度与嵌入式实时性；
2、公网 TCP 跨网联动：开发板拥有独立公网 IP，无需内网穿透，任意外网电脑可远程实时查看检测画面；
3、自适应可视化输出：推理画面自动缩放，检测文字放大加粗展示，适配嵌入式屏幕与上位机远程查看；
4、完整可复现全链路：从数据集训练、模型量化、端侧部署到上位机远程联动，全套开源代码无缺失。
